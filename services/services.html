C'est quoi ?


La vie d’un site web n’est pas calme : le Web est en constante évolution et de nouvelles tendances apparaissent régulièrement.


Par ailleurs, plusieurs situations portent à se pencher sur la refonte de son site web : un changement de stratégie de communication, de nouveaux objectifs webmarketing ou encore l’apparition de nouvelles tendances UX design.


La durée de vie d’un site web est estimée à 5 ans avant d’envisager une refonte.


Enfin, il est à noter que la refonte de site peut être conçue comme la création d'un site web, même s’il ne s’agit pas totalement du même travail.

Sites vitrines

Présenter une entreprise, une organisation, ou un produit. Le site vitrine s'inspire de l'univers du commerce de proximité.

Plus de détails >>


C'est quoi ?


Le site est dit statique ou vitrine car les pages web qui le composent sont toujours identiques entre deux visites des internautes.


Un site vitrine désigne un site web dont l’objectif est de décrire les activités d’une entreprise, d’une association, d’un organisme ou de toute structure.


En plus d’une page d’accueil, on y trouve souvent des rubriques pour présenter la structure, ses produits, ses services ou ses projets, ainsi qu’un formulaire de contact.


À la différence d’un site e-commerce, les sites vitrines ne servent pas à vendre mais à faire la promotion de la structure.

C'est quoi ?


Les grandes applications de ce type de site web sont : les forums, les Wiki (Wikipedia) et les sites communautaires (Facebook, Twitter, hi5, etc.).


Nous trouvons dans cette catégorie les CMS comme WordPress, Joomla, Drupal qui eux, offrent à nos clients la possibilité d'administrer eux-même leur site web.


Le serveur qui fait fonctionner le site utilise une technologie de scripting (comme Php, Ruby, Python ou Perl) ainsi qu'une base de données comme Mysql.


À la différence d’un site vitrine, les sites Dynamiques sont excellents pour les boutiques en ligne (e-commerce).
